Sentiment analysis of Danish texts (wordlist)

The process of determining the emotional tone behind a series of words, used to gain an understanding of the the attitudes, opinions and emotions

Sentiment analysis — also known as opinion mining — is as simple as assigning each word in a text a value based on the bias of the word, after which the overall sentiment is decided by summarizing the values.
This is especially useful for larger companies, who wish to gain an insight in their customers opionions regarding a certain topic. However, it could also be used as a tool for students analysing a Danish text, as it can help to quickly locate all positively or negatively charged words.

Disable spotify ads

Spotify is coded in such a way that it will keep playing music, if it can’t gaing access to an ad. This script redirects all Spotify’s requests to the address 0.0.0.0 at which there is no ad resources located; thus Spotify will never be able to reach and play an ad.
That’s all there is it to. Enjoy the music!

Get the code

Instructions

  • Copy the code from the gist and paste it into a bat file.
    (A bat file is a file with the file extension .bat)
  • Run it

<code snippet>

@echo 
title Spotify-noad
:: BatchGotAdmin
:-------------------------------------
REM  --> Check for permissions
   IF "%PROCESSOR_ARCHITECTURE%" EQU "amd64" (
>nul 2>&1 "%SYSTEMROOT%\SysWOW64\cacls.exe" "%SYSTEMROOT%\SysWOW64\config\system"
) ELSE (
>nul 2>&1 "%SYSTEMROOT%\system32\cacls.exe" "%SYSTEMROOT%\system32\config\system"
)
